## 📌 Overview

The **JEE & CET Percentile Predictor** is a fast, lightweight, and visually appealing web application that helps students estimate their:

- 📊 Percentile
- 🏆 Rank
- 🎯 Admission Chances
- 📈 Performance Analysis

based on their examination scores.

The application supports multiple national and state-level entrance examinations using historical trends and official result methodologies.

## 📊 Prediction Methodology

### JEE Main
Uses percentile normalization trends published by NTA and historical shift-wise distributions.

### JEE Advanced
Since official percentile data is unavailable, rank trends are transformed into percentile-strength estimates.

### MHT CET
Based on official CET Cell result processing methodologies and previous-year score distributions.

### KCET / GUJCET
Supports board-mark weightage adjustments where applicable.

### Other CET Examinations
Predictions are generated using historical marks-to-rank and percentile trend datasets.
